load_dask.load_dask module¶

load_dask.load_dask.load_dask(context: mlrun.execution.MLClientCtx, src_data: mlrun.datastore.base.DataItem, dask_key: str = 'dask_key', inc_cols: Optional[List[str]] = None, index_cols: Optional[List[str]] = None, dask_persist: bool = True, refresh_data: bool = True, scheduler_key: str = 'scheduler') → None[source]¶

Load dataset into an existing dask cluster

dask jobs define the dask client parameters at the job level, this method will raise an error if no client is detected.

Parameters

  • context – the function context

  • src_data – url of the data file or partitioned dataset as either artifact DataItem, string, or path object (similar to pandas read_csv)

  • dask_key – destination key of data on dask cluster and artifact store

  • inc_cols – include only these columns (very fast)

  • index_cols – list of index column names (can be a long-running process)

  • dask_persist – (True) should the data be persisted (through the client.persist op)

  • refresh_data – (False) if the dask_key already exists in the dask cluster, this will raise an Exception. Set to True to replace the existing cluster data.

  • scheduler_key – (scheduler) the dask scheduler configuration, json also logged as an artifact
